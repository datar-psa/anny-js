anny-js

A browser-native JavaScript runtime for Anny — Naver's open-source parametric 3D body model — plus a MediaPipe Pose + Hand driver that maps live landmarks onto Anny's 163-bone rig. No server, no Python, no native modules.

Anny in JS

Forward kinematics + linear blend skinning ported from Python anny to TypeScript. Parity-tested: < 1e-5 on bone transforms, < 0.1 mm on vertices.

MediaPipe → Anny

33 pose + 21×2 hand landmarks per frame → 163 bone deltas. Parent-corrected, angle-clamped, proximity-based L/R hand assignment.
